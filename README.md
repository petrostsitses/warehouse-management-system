# Warehouse Management System

A simple inventory management system built with Python.  
The project was developed as part of my learning process in programming and focuses on managing products, storing data, and performing basic analysis.

---

## Features

- Add new products to inventory
- Remove products by ID
- Search products by name
- Update product information
- Display all inventory items
- Calculate total inventory value
- Save and load data using JSON
- Export and import data using CSV
- Sort and filter products
- Basic inventory analysis
- Data visualization (bar chart, pie chart, scatter plot)

---

## Technologies Used

- Python
- JSON / CSV file handling
- Matplotlib (for charts)
